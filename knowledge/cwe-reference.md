# Common CWE Reference

## Injection
- CWE-89: SQL Injection
- CWE-78: OS Command Injection
- CWE-79: Cross-site Scripting (XSS)
- CWE-94: Code Injection
- CWE-611: XML External Entity (XXE)
- CWE-917: Expression Language Injection

## Authentication & Session
- CWE-287: Improper Authentication
- CWE-307: Brute Force / Excessive Auth Attempts
- CWE-384: Session Fixation
- CWE-330: Insufficient Random Values
- CWE-521: Weak Password Requirements
- CWE-798: Hard-coded Credentials

## Access Control
- CWE-200: Exposure of Sensitive Information
- CWE-284: Improper Access Control
- CWE-352: Cross-Site Request Forgery (CSRF)
- CWE-639: IDOR / Authorization Bypass via User Key
- CWE-862: Missing Authorization

## Cryptography
- CWE-319: Cleartext Transmission of Sensitive Info
- CWE-326: Inadequate Encryption Strength
- CWE-327: Broken/Risky Cryptographic Algorithm
- CWE-916: Insufficient Password Hashing

## Input Validation
- CWE-20: Improper Input Validation
- CWE-22: Path Traversal
- CWE-434: Unrestricted File Upload
- CWE-601: Open Redirect

## Network / Server
- CWE-918: SSRF
- CWE-295: Improper Certificate Validation
- CWE-209: Sensitive Info in Error Messages
